# Field Monitoring — Procedures

## How to Register a New Plot

1. Open **Field Management > Plots**
2. Click the **New Plot** button (green "+" icon, top right)
3. Fill in the required fields:
   - **Property**: Select parent property
   - **Plot Name**: Unique identifier
   - **Total Area (ha)**: In hectares
   - **Planting Year**: Year of vineyard establishment
4. Optional fields:
   - **Slope** and **Altitude**: For precision agriculture
   - **Row Spacing** / **Inter-row Spacing**: For plant density calculation
   - **Training System**: Select from dropdown
   - **Trellising Type**: Select from dropdown
5. Click **Save**

## How to Assign Crop Varieties

1. Open the plot detail screen
2. Navigate to the **Crops** tab
3. Click **Add Crop**
4. Select:
   - **Crop Type** (e.g., Vine)
   - **Variety** (e.g., Touriga Nacional)
   - **Area** dedicated to this variety
   - **Production Mode** (Integrated Production, Organic, etc.)
5. Click **Save**

## How to Configure Irrigation

1. Open **Field Management > Irrigation**
2. Select the plot
3. Choose **Irrigation Type** (Drip, Sprinkler, etc.)
4. Set the **Water Source** details
5. Save configuration

## Onde encontrar na aplicacao

- Plot registration: **Field Management > Plots > New**
- Varieties: **Field Management > Crop Registry**
- Irrigation: **Field Management > Irrigation**
