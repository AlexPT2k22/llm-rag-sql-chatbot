# Cellar Management Module

## Overview

The Cellar Management module handles all winemaking operations from grape intake to bottling,
including lot tracking, volume management, consumables, and human resources.

## Grape Intake

The **Intake** process registers grape arrivals:
- Weight (kg)
- Origin plot
- Grape variety
- Transport details
- Quality analysis results

## Lots and Tanks

### Lots
A **lot** represents a tracked wine batch:
- Lot code (e.g., DOC Red, Vinho Verde White)
- Wine type and category
- Current phase (fermentation, aging, bottling)
- Total volume (liters)

### Tanks
**Tanks** (or barrels) store wine during production:
- Capacity and current contents
- Material (stainless steel, oak barrel)
- Location in cellar

## Winemaking Operations

All cellar operations are tracked:
- Crushing and pressing
- Fermentation monitoring
- Racking and filtration
- Stabilization
- Bottling and labeling
- Sample analysis

## Cellar Views

- **Operations**: Operations per lot and tank with costs and hours
- **Volumes**: Current wine volumes by type and phase
- **Losses**: Wine losses by phase
- **HR Calendar**: Monthly worker hours in cellar operations
- **Consumables**: Consumable product usage (bottles, corks, labels)

## Onde encontrar na aplicacao

1. Navigate to **Cellar Management** in the sidebar
2. Use **Intake** for grape reception
3. **Lots** for lot tracking
4. **Operations** for winemaking tasks
5. **Inventory** for volumes and consumables
